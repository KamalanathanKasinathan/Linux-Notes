1) cat

cat (concatenate) command allows us to create single file, view contents of file, concatenate files and redirect output in terminal or files.

Example of creating file or files using cat command:

# cat > filename

To create hidden file:

# cat > .file

2) touch

The touch command is the easiest way to create new, empty files that is 0kb file.

Example of creating file or files using touch command:

To create an empty file

# touch file1

To create multiple empty files

# touch file1 file2 file3 file4
# touch file{1..4}

To create hidden empty file

# touch .hiddenfile

1) vi (Visual Instrument)

A screen-oriented text editor originally created for the Unix operating system.

# vi filename
# vi .filename

2) vim (Visual Instrument iMproved)

The vim editor is an enhanced version of vi.

# yum install vim-enhanced -y
# vim filename
# vim .filename

3) nano

This is a command-line notepad.

# nano filename

What is the Difference Between Vim and Vi?

Vim: Basically 'vim' is a text editor which opens a blank file which you can edit and save it. By using vim editor, file content can be encrypted.

Vi: File content cannot be encrypted.

Vi, Vim, Gvim Editor Modes

  • Command Mode - press Esc (default mode)
  • Insert Mode - press i, I, A, a, o, O
  • Execution Mode (or Escape Mode) - : (shift + ;)

Command Mode (default mode)

yy --> to copy a line
3yy --> to copy three lines
p --> to paste below the cursor
P --> to paste above the cursor
4p --> to paste copied line 4 times
dd --> to delete a line
4dd --> to delete 4 lines
shift + g(G) --> moves cursor to end of file
gg --> moves cursor to beginning of file
/word --> to search a keyword
n --> next search
N --> previous search
u --> undo
ctrl + r --> redo

To Move Cursor in Command Mode

h - move left
j - move down
k - move up
l - move right

To Save and Quit from Command Mode

shift + zz

Insert Mode

i --> insert text
I --> insert text at beginning of line
a --> add text after cursor
A --> add text at end of line
o --> insert text one line below cursor
O --> insert text one line above cursor

Ex Mode (Extended Command Mode)

:q --> quit without saving
:q! --> force quit
:w --> save changes
:w! --> force save changes
:wq --> save and quit
:wq! --> force save and quit
:x --> save and quit
:X --> encrypt file contents
:25 --> move cursor to 25th line
:set nu --> set line numbers
:set nonu --> remove line numbers
:5d --> delete 5th line
:$d --> delete last line
:3,6d --> delete lines 3 to 6
:$ --> move cursor to last line

To Search a Word and Replace It

:%s/old/new/g

Without Opening a File - Find and Replace a Word

# sed -e "s/old/new/g" filename
