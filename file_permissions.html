Linux File Permissions

Types of Permissions

  • Basic file/folder permissions
  • Special file/folder permissions
  • Access control list (ACL)

Basic File/Folder Permissions

Checking Default Permissions

    # ls -l
    # ll

Fields in ls -l Command Output

<file type> <file permission> <link count> <owner> <group> <file size> <last modify date and time> <file name>

File Types and Permissions

File Type User Group Others
d (Directory) rwx r-x r-x
- (Regular File) rw- r-- r--
l (Symbolic Link) rwx rwx rwx

ls Command Options

    # ls -iadRrlt
  • i - List the file's inode number
  • a - List hidden files and folders
  • d - List only directories
  • R - List the tree structure of a directory
  • l - List all information about files and folders
  • r - List files and folders in reverse order
  • t - List files and folders according to timestamp

File Permissions

Read: Permission to read file content

Write: Permission to modify content

Execute: Permission to run script

Directory Permissions

Read: Permission to list contents (ls)

Write: Permission to create or delete files

Execute: Permission to change directory

Changing Permissions

    # chmod (permission value)

Using two methods:

  • Octal (Numeric) - 421
  • Symbolic - rwx

Numeric Method Examples

    # chmod 744 
    # chmod 440 
    # chmod 244

Symbolic Method Examples

    # chmod u=rwx,g=rw,o=r 
    # chmod u-x,g-rw,o+r 
    # chmod a=rwx testfile1
    # chmod a-wx testfile1
    # chmod a+wx testfile

Default Permissions

Normal User

  • 775 - Directory
  • 664 - File

Root User

  • 755 - Directory
  • 644 - File

umask - File Creation Mask

Controls default file permissions.

    umask value for root user: -022
    umask value for normal user: -002
