Common Linux Log Files

/var/log/messages

Primary log file that records system events, including boot messages and system errors.

/var/log/secure

Tracks security-related events like authentication attempts and sudo usage.

/var/log/cron

Logs generated by the cron daemon for scheduled tasks.

/var/log/dmesg

Kernel ring buffer messages useful for diagnosing hardware and driver issues.

/var/log/boot.log

Captures messages related to the system boot process.

/var/log/yum.log

Tracks package installation, updates, and removals performed using YUM.

/var/log/audit/audit.log

Stores logs from the Linux Auditing System for security monitoring.

/var/log/httpd/

Contains Apache web server logs, including access and error logs.

/var/log/maillog

Logs related to the mail server, including sent and received messages.

/var/log/lastlog

Records the last login time of each user; can be viewed using the lastlog command.

/var/log/btmp

Records failed login attempts; can be viewed using the lastb command.

/var/log/wtmp

Records login and logout events; can be viewed using the last command.
