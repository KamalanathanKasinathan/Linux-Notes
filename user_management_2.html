User and Group Management Commands

To add a user into a group

# useradd -g groupname username

useradd with predefined parameters

# useradd -u 2000 -g 3000 -c 'test user' -d /home/test1 -s /bin/sh username
# useradd -M
# useradd -e '2021-12-25' username

Options:

  • -l : login name
  • -u : user ID (UID)
  • -g : group ID (GID)
  • -c : user comment
  • -d : home directory
  • -s : shell

usermod Commands

To rename a user's account name

# usermod -l newname oldname

To change user's UID

# usermod -u 1076 username

To change user's default home directory

# usermod -d /new/directory username

To change the default shell

# usermod -s /bin/sh username

Setting a User Expiry Date

# usermod -e "2022-02-21" username

Locking and Unlocking a User Account

Lock:

# usermod -L USER

Unlock:

# usermod -U USER

Checking User and Group Information

# id username  (Check User ID and Group ID)
# chage -l username  (Verify User's Password Info)

Password Expiry Management

# chage -m 5 username  (Set minimum password expiry to 5 days)
# chage -M 20 username (Set maximum password expiry to 20 days)
# chage -W 5 username  (Set password warning period to 5 days)
# chage -I 5 username  (Set password inactive date to 5 days)

To force a password change after first login

# chage -d 0 username

Group Management

To create a group

# groupadd groupname

To add a user into a group

# gpasswd -a user group
# usermod -a -G group user

To change a user's GID

# groupmod -g GID username

To change a group's GID

# groupmod -g GID groupname

To rename a group

# groupmod -n newgroup oldgroup

Lock and Unlock User Password

# passwd -l username (Lock password)
# passwd -u username (Unlock password)
